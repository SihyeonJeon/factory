## P0 — 근본적 방향 수정

### HF-1: 앱 이름 + 브랜딩
- 앱 이름은 "Unfading"이다. "Memory Map" 아님.
- 의미: "우리의 흐려지지 않는 추억"
- 모든 타이틀, 배너, 내부 문구에서 Unfading으로 변경

### HF-2: 전체 한글화
- **모든** UI 텍스트를 한글로 변경
- 버튼, 라벨, placeholder, 에러 메시지, 탭 이름, 섹션 헤더 전부
- 현재 영어로 된 모든 것: "Save", "Cancel", "New Memory", "Add from Library", 등

### HF-3: 디자인 톤 변경
- 현재: 딱딱한 애플 리퀴드 글래스 느낌
- 원하는 것: 귀엽고 따뜻한 한국 20~30대/커플 감성
- 둥근 모서리, 부드러운 파스텔 색상, 따뜻한 타이포그래피
- 참고: 한국에서 인기 있는 커플 다이어리/사진 앱 톤

### HF-4: 상단 검색 바 추가
- 현재: "Memory Map" 배너만 있음
- 원하는 것: 장소 검색 가능한 검색 바가 상단에 위치

## P1 — 핵심 인터랙션 수정

### HF-5: 바텀 시트 스냅 불자연스러움
- 손을 떼면 액션이 한 번 더 발생
- 자연스러운 스냅 + 바운스가 필요

### HF-6: 마커 클러스터링 미작동
- 지도 축소해도 마커들이 합쳐지지 않음
- Google Maps 저장 장소처럼: 마커가 일정 거리 이하에서 자동 합쳐짐
- MKClusterAnnotation 또는 커스텀 클러스터링 필요

### HF-7: 바텀 시트 내 추억 보관함 전면 재설계
상세 스펙:
- 기본 상태: 큐레이션 + 추억 보관함
- 사진첩/갤러리처럼: 데이트 이름 하위에 사진 정사각형 그리드
- 사진 탭 → 바텀 시트 내 "추억 간략히 보기" 진입 + 뒤로가기 버튼 + 지도 해당 마커 선택/확대
- 지도에서 마커 선택 → 같은 "추억 간략히 보기"로 이동
- 추억 간략히 보기 (바텀 시트 중간): 간단한 내용 조회
- 스크롤 내리면 → 바텀 시트가 전체화면으로 확대 → "추억 상세 페이지"로 전환
- 추억 상세: 핸들 숨김 (보이지 않을 뿐 기능은 유지), 새 페이지 느낌
- 내리거나 뒤로가기 → 바텀 시트 50% 복원
- 좌우 화살표로 앞뒤 추억 이동
- 클러스터 마커 선택 시: 해당 추억들만 필터링 노출
- 여러 데이트에 걸친 추억: 각 데이트별 그룹핑

## P2 — 개선 요청

### HF-8: Google Maps 전환
- Apple Maps 테마가 예쁘지 않음
- Google Maps SDK 사용 가능한지 검토

### HF-9: 전반적 UI/UX 어색함
- 버튼 위치, 글자 배치 등 어색한 곳 다수
- 기능이 어디 있는지 모르겠음
- → 전체 화면 레이아웃 리뷰 필요

### HF-10: Supabase MCP 연동
- 데이터베이스 제어/수정을 MCP로 진행
- → 인프라 설정 필요, 하네스와 별도
