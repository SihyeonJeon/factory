# Colab Payloads

These files are designed for the `colab-mcp` execution path.

Payloads are standalone scripts or cell sequences intended to be pasted into a Colab cell or saved into the Colab runtime filesystem and executed there.

This path exists because `colab-mcp` bridges a local agent to a browser Colab session, but it does not automatically mirror the local repository filesystem into the remote runtime.
