# Moment MVP — 수용 기준 (Acceptance Criteria)

## Epic 1: 이벤트 생성 + 카카오톡 공유

| ID | 기준 | 검증 방법 |
|----|------|----------|
| E1-01 | 4단계 위자드 (무드→커버→정보→미리보기) 완성 | UI 테스트 |
| E1-02 | 6개 무드 테마, 각 테마별 고유 색상 + 기본 커버 | 시각적 확인 |
| E1-03 | 커버 이미지 업로드 (JPEG/PNG/WebP/HEIC, 10MB 제한) | 파일 업로드 테스트 |
| E1-04 | 이벤트 URL 생성 후 공유 가능 | URL 접근 테스트 |
| E1-05 | 카카오톡 OG 미리보기: 제목 + 날짜 + 장소 + 무드 이모지 | 카카오톡 공유 테스트 |
| E1-06 | OG 이미지 1200×630, 한국어 정상 렌더링 | 이미지 검증 |

## Epic 2: PWA RSVP

| ID | 기준 | 검증 방법 |
|----|------|----------|

| E2-04 | 비로그인 → 로그인 리다이렉트 → 원래 페이지 복귀 | 플로우 테스트 |
| E2-05 | PWA 매니페스트 + PNG 아이콘 (192, 512) + 설치 프롬프트 | PWA 검증 |
| E2-06 | 응답 변경 가능 | UI 테스트 |
